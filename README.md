# 📊 Análise de Dados - Empresa Fictícia

## 📌 Sobre o projeto

Este projeto realiza uma análise exploratória de dados de uma empresa fictícia, com o objetivo de identificar padrões de vendas, desempenho de produtos e insights para tomada de decisão.

---

## 🚀 Tecnologias utilizadas

Este projeto foi desenvolvido utilizando:

- Python — linguagem principal do projeto  
- Pandas — análise e manipulação de dados  
- Plotly Express — criação de gráficos interativos e visualizações  

---

## 📊 O que foi analisado

- Distribuição de clientes por plano:
  - Básico  
  - Standard  
  - Premium  
  - Anual  
  - Mensal  

- Comparação entre clientes ativos e cancelados  
- Taxa de cancelamento (churn)  
- Relação entre tipo de plano e cancelamento  
- Possíveis causas do cancelamento

---

## 📈 Principais insights

- Foi identificada diferença significativa na taxa de cancelamento entre os planos  
- Alguns planos apresentam maior retenção de clientes  
- Clientes de determinados planos têm maior tendência a cancelar  
- Foram sugeridas estratégias para reduzir a taxa de churn    

---

## 💡 Possíveis melhorias sugeridas

- Melhorar benefícios dos planos com maior taxa de cancelamento  
- Criar campanhas de retenção para clientes em risco  
- Ajustar preços ou benefícios de planos menos utilizados  
- Analisar comportamento antes do cancelamento
